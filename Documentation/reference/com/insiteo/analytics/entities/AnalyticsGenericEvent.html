AnalyticsGenericEvent

com.insiteo.analytics.entities.AnalyticsGenericEvent

Class Overview

Generic event that can be used to catch specific behavior. This event will be sort by its type and can contain 2 String parameters, 2 int parameters, 2 double parameters and 2 Positions. All the required field of the event must be filled before passing it to the AnalyticsManager.
