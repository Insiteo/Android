public IRTO cloneRTO ()

Since: API Level

Create a copy of this RTO. The cloned RTO shall have the same ID as the original. This method should copy the behavior of Object.clone() method. It is called when putting a single RTO in multiple zones : thus, the cloned RTO can share resources - such as bitmaps - with the original.

Returns
  • a new RTO that is the exact copy of the original, or null if object could not be cloned.

public void draw (Canvas aCanvas, double aRatio, Point aOffset, float aAngle)

Since: API Level

Draws this object on the given Canvas. To obtain object position on screen, do (object_position_in_meter * aRatio + aOffset). If this object has no position (ie : IRTO is put in a zone), simply draw it at aOffset.

Parameters
aCanvas The destination Canvas where this object should be drawn to.
aRatio The real/screen coord ratio, this is used to transform the object's real coords into screen coords.
aOffset In pixels, the offset from the (0, 0) of the drawing Canvas that should be added to this object coords.
aAngle The offset angle in case a rotation should be done. The rotation is relative to the screen center.

public void draw3D (World world, Map map, double ratio, float angle)

Since: API Level

Draws this object with its 3D mode on the given Canvas. To obtain object position on screen, do (object_position_in_meter * aRatio + aOffset). If this object has no position (ie : IRTO is put in a zone), simply draw it at aOffset.

Parameters
world The JPCT current World.
map The current map.
ratio The ratio to apply to meter coordinates to transform them in screen coordinates.
angle The angle between screen's y axis and the canvas y axis.
