Class Overview

This class handles the geofencing process. You can retrieve an instance of it by calling (GeofenceProvider)LocationProvider.getInstance().getModule(ELocationModule.GEOFENCING). It provides a way to be notified when location is detected in a certain area (see GeofenceZone), defined on Insiteo servers. To use this class, you just have to call setListener(IGeofenceListener), and start the location. The listener will then receive notification each time location enter/stay/leave an area.
