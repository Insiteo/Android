public abstract void onDataUpdateDone (boolean aSuccess, InsiteoError aError)

Since: API Level

This will be called when package update is finished (download + install).

Parameters
aSuccess the status of update : true if update was successful, false otherwise
aError an error associated to the status. This error contains a code, and a either a message (as a string) or a messageId (to get message from resources strings).

public abstract void onDownloadProgress (long aDownloadedBytes, long aTotalBytes)

Since: API Level

This will be called during package download, and can be used to fill a progress bar. This will be called on UI thread.

Parameters
aDownloadedBytes the number of bytes already downloaded for the current package
aTotalBytes the size of the package, in byte

public abstract void onInitDone (EInitResult aResult, InsiteoError aError)

Since: API Level

This will be called once API initialization is done.

Parameters
aResult the status of initialization
  • SUCCESS if init was successful
  • SUCCESS_WITH_NEW_DATA if init was successful and new data packages are available on server (call InitProvider.updatePackages to download new data)
  • FAIL otherwise (call InitProvider.hasPackage to check data availability for each module you want to use)
aError an InsiteoError if init failed with known reason, null otherwise.

public abstract void onInstallProgress (long aCurrentFile, long aTotalFiles)

Since: API Level

aCurrentFile the number of files already deployed
aTotalFiles the number of file to deploy
