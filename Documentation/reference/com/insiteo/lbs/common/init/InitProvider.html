Class Overview

Singleton that handles initialization of Insiteo API. It will store useful information such as data version, or URLs for requests. Be sure to call #initAPI(Activity, IInitListener, Package, boolean, boolean, String, int, int, String) init before using other insiteo modules.

public ICancelable startAPI (Activity aContext, IInitListener aListener, String aInitUrl, int aSiteId, int aAppVersion, String aLang, InitProvider.Server aServer, boolean aAnalyticsAutoStart)

Since: API Level

Request for API initialization. This is an asynchronous call : results will be send via aListener's callbacks. This method will return an ICancelable that allow you to stop the background init task.

Parameters
aContext the context in which init will be done
aListener the listener that will receive init notifications. The listener is stored as a WeakReference : be careful to keep a reference on it in your code if you want to receive notifications.
aInitUrl the url of the init request
aSiteId the site ID of the calling application
aAppVersion the application code version of the calling application
aLang the locale of the calling application
aServer the server the application is used on
Returns
  • an ICancelable object, representing the init task that is running background. Call cancel() on it to stop the init task (before you kill aContext for example).

public ICancelable updatePackages (Activity aContext, IInitListener aListener, boolean aForceUpdate)

Since: API Level

Update all data for insiteo modules. This method will download and install packages containing all available data for the site defined at init. Download and install are running asynchronously : aListener will be notified of progress.

Parameters
aContext the context in which this task will run.
aListener the listener that will receive progress notification. If null, you won't receive notification.
aForceUpdate if false, only the new packages will be installed ; if true all packages will be downloaded and installed, even if they are already deployed
Returns
  • an ICancelable object. Call cancel() on this ICancelable to abort this update (no notification will be sent after cancel).
