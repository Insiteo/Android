Class Overview

Class containing Insiteo error information. Use toString(Resources) or toString() to get the message associated to this error. It contains :

  • a code, determining the error type
  • a message, containing a readable string describing the error
  • a messageId : if different from NULL_ID, it gives the id of a string resource containing localized error description
If the InsiteoError contains a message, it will not contain a messageId. If the InsiteoError contains a messageId, it will not contain a message.
