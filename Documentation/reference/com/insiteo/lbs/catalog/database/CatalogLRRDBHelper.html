public ArrayList<CatalogEntry> searchEntries (String aSearchedText, boolean aSplitWords, int aPoiTypeID, String[] aCategoryColumns, String[] aPoiColumns)

Since: API Level

Method called to execute a text search and get all related entries (Category and Poi).

Parameters
aSearchedText The text searched in database fields.
aSplitWords if false, the searchedText parameter is used as a unique string. If true we split this string in several words (space separated)
aPoiTypeID A poi type used to filter the results. Set it to CommonConstants.NULL_ID if not needed.
aCategoryColumns the columns in the category table to search for. If null the request will be done on all columns.
aPoiColumns the columns in the poi table to search for. If null the request will be done on all columns.
