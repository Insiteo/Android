AnalyticsUserEvent

com.insiteo.lbs.analytics.entities.AnalyticsUserEvent

Class Overview

User event that can be used to trace the type of device (OS version, resolution, carrier, network, device name and model) and Insiteo site identificators (site id and lang, application and API version).
