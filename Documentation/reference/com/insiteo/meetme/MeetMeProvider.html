Class Overview

This class is the entry point for MeetMe service. You can retrieve an instance of it by calling (MeetMeProvider)LocationProvider.getInstance().getModule(ELocationModule.MEETME). It handles requests with server, and notify its listener when new data is available. Groups management can be done using external events (in this case, events are managed by application, using APIs such as facebook or linkedin). It can also be done using Insiteo events (in this case, MeetMeProvider provides methods to create insiteo events). Login can be done using external profile (managed by application, using APIs such as facebook or linkedin). It can also be done using insiteo profile :

  • either with a permanent profile (login + password + nickname)
  • or with a temporary profile, using only a nickname
