IMapData

com.insiteo.common.IMapData
Known Indirect Subclasses
MapData
MapData Class containing map data needed to display a map, such as map's scale, zones color, map's bounds etc. 

Class Overview

Interface used to describe map's data. Instances of this interface can perform conversion from map's coordinates to lat-long if GeoMatrix is filled.
