public boolean start (Context aContext, ILocationListener aListener, int aLocationFlags, long aScanRefreshDelay, long aScanRefreshDelayBackground, long aBackgroundDuration, int aDefaultMapID, boolean aForceDefaultMap, String locationDirPath)

Since: API Level

Start the location provider only if it is not already started. To change the location flags see changeLocation method. This method is asynchronous the initialization result will be notified via the onLocationInitDone(boolean, InsiteoError).

Parameters
aContext the context used to start the location service
aListener the listener that will receive location notifications. The listener is stored as a WeakReference : be careful to keep a reference on it in your code if you want to receive notifications.
aLocationFlags a flag indicating what type of location to use
aScanRefreshDelay (milliseconds) Delay between two wifi scans when location is running foreground. When set to 0, scans will run as fast as possible.
aScanRefreshDelayBackground (milliseconds) Delay between two wifi scans when location is running background. When set to 0, scans will run as fast as possible.
aBackgroundDuration (milliseconds) When stopped, location will continue running background for this amount of time.
aDefaultMapID the ID of the map used by default for location (if only GPS data is available for example). If set to -1, it is ignored.
aForceDefaultMap if true, location will be forced on default map
Returns
  • true if the location provider was started successfully (or was already started)
